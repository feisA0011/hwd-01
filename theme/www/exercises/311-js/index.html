Exercise 311 - JS, Event fun

This exercise is about using events to create effects.

The purpose of this exercise is to practice basic JS, by capturing and dealing with events and using them to do something completely different, than the user would probably expect.

Use the script tag in the bottom of the exercise page to write your JavaScript.

Tasks

  1. Add a wheel event listener to the Window object

    This event will occur when the user uses the scroll wheel on their mouse or a trackpad with scrolling capabilities on their computer.

  2. Detect scroll direction

    The page won't actually scroll, but the event will still occur with a delta setting, which tells you the direction of the scroll.

  3. Deal with the delta value

    Use the delta value to increase or decrease the font size of the text.
