Exercise 310 - HTML form

This exercise is about using native Form and input events.

The purpose of this exercise is to practice basic JS, by capturing and dealing with form and input events. You will be using the focus, blur, update and submit events in this exercise.

Use the script tag in the bottom of the exercise page to write your JavaScript.

Tasks

  1. Add eventlisteners to inputs

    Add a focus, blur and update eventlistener to each of the input fields in the exercise page.

  2. Add eventlistener to form

    Add a submit eventlistener to the form in the exercise page.

  3. Deal with focus and blur events

    On focus you add a "focus" class to the div.field containing the input. Correct application will make the border of the input orange. On blur you remove the "focus" class again.

  4. Deal with update events

    On update you copy the content of the input field into a div, that you create solely for the purpose of mirroring the current value of the input field. Create on div for each input.

  5. Deal with submit events

    On submit you create an alert to inform that the form has been submitted. You can make the submit event callback function return false, to prevent the form for actually being submitted.
