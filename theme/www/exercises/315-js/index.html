Exercise 315 - JS, Manipulator library 2

This exercise is about using the Manipulator library.

The purpose of this exercise is to practice using a combination of different tools to solve a challenge.

Use the script tag in the bottom of the exercise page to write your JavaScript.

The manipulator library is included in this page. You can use everything you find in the Manipulator library documentation.

Tasks

  1. Build a DOM structure

    Add a structure containing a div with the classname scene. Add it to div#content. Make the scene as high as the browser window.

    Add three div's to the scene.

  2. Request quotes

    Use u.request to make a request to https://random-quote-generator.herokuapp.com/api/quotes/random to get a quote for each of the three div's. You are supposed to make three requests so you get a different quote for each div. The Quote API returns a JSON object â€“ you should use the quote value, to insert into your div.

  3. Add CSS

    Use u.ass to make the 3 div's 100px wide and high. You may distribute them nicely if you wish.

    Add three different colors and/or background animations to the three divs. Background animations can ba created by adding additional html as needed.

  4. Add event listeners

    Add a click listener to each of the three div's. Prepare them to make an event callback to the same function, which will make them jump.

    In the event callback, make the clicked element jump to the bottom of the screen by applying a CSS translate property. Wait for one second, then let it jump back up. Use u.a.transition to apply transition timing to the movement. Use the third parameter to add a custom callback method where you apply the "jump back" animation.
