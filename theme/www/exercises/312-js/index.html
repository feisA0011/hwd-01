Exercise 312 - JS, Requests

This exercise is about using XMLHttpRequest (XHR) to load data from external sources.

The purpose of this exercise is to practice basic JS, by making a request and injecting part of the response back into the DOM.

Use the script tag in the bottom of the exercise page to write your JavaScript.

Tasks

  1. Create a new XHR request object

    Create a new XMLHttpRequest instance, and define a readyState change handler for your new instance.

  2. Make a request

    Use the XHR.open and XHR.send methods to make a request to http://hwd-01.local. Use the XHR.responseType and set it to document to return the response as Document object.

  3. Deal with the response

    When you receive the response in your readyState handler, then query the div#content element in the response object, and inject the content into the div#content element in your page.
