Exercise 307 - JS, JSON to HTML

This exercise is about using native JSON as a data source.

The purpose of this exercise is to practice basic JS, by translating a JSON object into an HTML structure. The JSON object contains information, which you should use to build an HTML structure using JavaScript.

Use the script tag in the bottom of the exercise page to write your JavaScript.

Tasks

  1. Query the #content element

    Make a query to get the #content. This node is the injection point for the actual exercise.

  2. Create and append a new ul element

    Use JavaScript to create a new ul element and insert it into the #content element. Keep a reference to the new ul.

  3. Make a for...in loop

    Make a for...in loop to iterate the JSON object, stored in content_set. For each iteration you should add a new li, using the key as classname and the value as text, to your newly created ul.
