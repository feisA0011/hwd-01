Exercise 319 - JS, Reverse engineering

This exercise is about reading code.

The purpose of this exercise is to practice reading code. You have the read the JavaScript code and figure out what kind of HTML it needs to work.

Tasks

  1. Read the code

    Read through the JavaScript in the tag in the bottom of the exercise page.

  2. Create the HTML

    From the code you now have to deduct what HTML elements are required for the script to run without errors. Write the HTML into the div#content as regular HTML.
