HWD: Exercises

Below you will find a list of exercises and puzzles for training and expanding your coding skills.

General instructions

Take the time to fully read these instructions before you begin.

Exercise content

All exercises contain the following two elements:

  • An index.html - an HTML file containing the exercise instructions.
  • A material folder - a folder named with the exercise number, containing the material needed for the exercise.

Before you start

Make a new personal exercise folder in www/homework with your name.

When you start a new exercise, copy the material folder into your personal exercise folder.

For Exercise 001, copy the folder 001 from www/exercises/001-html to your personal exercise folder. See www/homework/mr_miyagi to see an example.

About skipping exercises

Wax on, wax off. Don't skip.

You should take the time to complete all exercises in the listed order, rather than skipping ahead.

The exercises are intended to be completed in the order listed below because they occasionally build on each other.

If you feel that you are already a master of a topic and don't need more training, then get off your horse and train some more.
