Exercise 304 - JS, DOM manipulation

This exercise is about using native JS for manipulating the DOM by adding/removing classes.

The purpose of this exercise is to practice basic JS, by querying the relevant Elements and creating a loop with a conditional statement to identify whether to add or remove a class.

Use the script tag in the bottom of the exercise page to write your JavaScript.

Tasks

  1. Query all li's in ul.modifyme

    Make a query to get a nodelist of all the li's in ul.modifyme. Do not use the :not selector - that is not the purpose of this exercise.

  2. Make a loop to iterate the nodelist

    Make a regular for-loop that iterates the nodelist, so you have a way to inspect each node.

  3. Add a conditional statement (if [else]) to the loop

    Check if a node has a special class. If it does, then remove the special class. If it does not have a special class, then add a new modified class.

Hints

  • The result should have 4 green li's and no red li's in the first list. The second list should be unchanged.
