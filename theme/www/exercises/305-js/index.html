Exercise 305 - JS, DOM traversing by iteration

This exercise is about using native JS for iterating the DOM.

The purpose of this exercise is to practice basic JS, by traversing the DOM using a while loop. The loop should help you travel from the starting point (specified below), to the top of the dom, represented by the document.body element.

Use the script tag in the bottom of the exercise page to write your JavaScript.

Tasks

  1. Query the .starthere element

    Make a query to get the li.starthere element. This node is the starting point for the actual exercise.

  2. Make a loop to iterate to the top of the page

    Make a while-loop. For each iteration you should check if you have arrived at the destination and if not, then continue to the next parent node.

  3. Output the final node

    Make a console.log output of the final node, which should be the document.body.

Hints

  • Use console.log to keep track of the process.
