Exercise 314 - JS, Manipulator library 1

This exercise is about using Manipulator to build a DOM structure and add event listeners.

The exercise expects you to have finished exercise 313.

The purpose of this exercise is to practice using a JavaScript library, by creating and manipulating the DOM with the Manipulator library.

Use the script tag in the bottom of the exercise page to write your JavaScript.

The manipulator library is included in this page. You can use everything you find in the Manipulator library documentation.

Tasks

  1. Build a DOM structure

    Use u.ae to build a structure containing a div with the classname scene. Add it to div#content.

    Add a heading, two paragraphs and a list with three list items.

  2. Add event listeners

    Use u.e.click to add an event listener to the heading, and let it toggle between four colors when you click on it. So each time you click the heading it changes to a new color.

    Use u.e.hold to add an event listener to the paragraph, and let it make an alert when you hold the mouse down for 2 seconds. Use the setting for u.e.hold to specify the desired hold time.

    Use u.e.hover to add an event listener to each of the three list items, and make it so the two other list elements change color, when you hover the mouse over each of them.
