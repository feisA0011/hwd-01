Exercise 301 - JS, Query and manipulate

This exercise is about using native JS for querying and manipulation the HTML.

The purpose of this exercise is to practice basic DOM manipulation, by querying elements and moving them into correct order.

Use the script tag in the bottom of the exercise page to write your JavaScript.
