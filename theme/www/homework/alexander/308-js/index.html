Exercise 308 - JS, Browser events

This exercise is about using native Browser events.

The purpose of this exercise is to practice basic JS, by capturing and dealing with events from by the browser. You will be using the resize, scroll and DOMContentLoaded events in this exercise.

Use the script tag in the bottom of the exercise page to write your JavaScript.

Tasks

  1. Add a DOMContentLoaded event listener to the Window object

    Add a DOMContentLoaded event listener to the Window object. Make a function called contentLoaded and use this as the callback function for the event listener.

    In contentLoaded, create a new div with the text "Content loaded", and append it to #content, to show that the content has loaded.

  2. Add a resize and scroll event listener.

    When the content has loaded - and only after this event has occurred - then add event listeners for resize and scroll events.

  3. Deal with resize events

    Find a way to identify the new width and height of the browser. Create one new div in #content and write the current size of the window into this div. You should not create a new div for each event, but rather re-use the same div for all resize events, so that it always contains the current size of the window.

  4. Deal with scroll events

    Find a way to identify the current scroll position of the browser. Create one new div in #content and write the current scroll position into this div. You should not create a new div for each event, but rather re-use the same div for all scroll events, so that it always contains the current scroll position.
