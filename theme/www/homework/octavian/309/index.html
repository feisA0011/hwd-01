Exercise 309 - JS, Mouse events

This exercise is about using native Mouse events.

The purpose of this exercise is to practice basic JS, by capturing and dealing with mouse events. You will be using the mouseover, mouseout and click events in this exercise.

Use the script tag in the bottom of the exercise page to write your JavaScript.

Tasks

  1. Query the li's of the navigation

    Get a nodelist with the li-elements of the navigation.

  2. Make a for loop and add event listeners

    Make a for loop to iterate the navigation elements. On each element you add a mouseover, a mouseout and a click event listener - to the li-element.

  3. Deal with mouseover events

    On mouseover you have to add a "hover" class to the element that the mouse hovered over. This will apply a new style to the element.

    4. Deal with mouseout events

    On mouseout you have to remove the "hover" class from the element that the mouse now left. This will return the element to it's original state.

  4. Deal with click events

    On click, you should make an alert with the text of the element the user clicked on.
